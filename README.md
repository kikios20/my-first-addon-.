# Assault Rifle Mod

## Описание
Мод добавляет в игру автомат с системой патронов и магазинов.

## Предметы
- **Assault Rifle (M16)** - основное оружие
- **Ammunition Magazine** - магазин с патронами
- **Bullet** - боеприпасы

## Механика
1. Возьми автомат и магазин
2. Нажми ПКМ с магазином в руке на автомат для перезарядки
3. Нажми ЛКМ с автоматом для стрельбы
4. Звуки и визуальные эффекты при выстреле
5. После 30 выстрелов нужна перезарядка

## Установка
Смотри инструкцию выше в README

## Версия
Подходит для Bedrock 1.20+
